# art-canvas :art:

Using [Zig](https://ziglang.org/) to compile a `.wasm` cart
for use in a [Canvas](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)

## Development

File watcher can be started by calling:
```sh
make spy
```

Running the canvas:
```sh
make run
```

Deploy:
```
make deploy
```
